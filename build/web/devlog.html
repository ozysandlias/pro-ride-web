Game Plan

This is my first post discussing pro-ride. I don't want to make any promises too big but I'd like to discuss what's in so far and where I plan to take it.

Current Features

  • Basic hoverboard handling
  • Full 360 degree control in air
  • Tumbling mode and glide mode toggleable by pressing jump
  • You can ride loop-de-loops
  • Half-Pipes/Ramps

Concrete Planned Features

  • Flip tricks and grab tricks
  • Grinding (still trying to decide if it's worth complicating the controls)
  • Bailing
  • Better Momentum Physics
  • Buffs that increase board stats, i.e. speed, acceleration, grip, air speed, tumbling, jumping
  • Different boards that handle in unique ways
  • Events that either lead to elimination or provide advantages
  • Timed mode with one competition at the end
  • Endless mode where the goal is to pass as many elimination events as you can

Features I'd like to implement but may require more studying

  • Items that can be used as weapons or movement tools
  • Online Multiplayer
  • Character customization
  • Clothing customization a la Animal Crossing
  • Map Editor
